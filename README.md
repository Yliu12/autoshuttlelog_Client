## Deploy Instruction
 [Install Node.js](https://nodejs.org/)


clone code

Install Angular CLI
```
npm install -g @angular/cli
```

Go to folder
```
cd autoshuttlelog_Client\
```
Install packages
```
npm i
```


Run Preview
```
ng serve
```


go to browser and open 

http://localhost:4200/driver
OR
http://localhost:4200/manager
